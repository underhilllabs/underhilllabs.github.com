Restart Apache2

{% highlight bash %} sudo apache2ctl graceful {% endhighlight %}

Continuously Monitor Apache Error Log

{% highlight bash %} sudo tail -f /var/log/apache2/error.log {% endhighlight %}

Apache Configuration

Add a new VirtualHost to the Apache Config file

{% highlight bash %} sudo vi /etc/apache2/sites-available/default {% endhighlight %}

A simple VirtualHost entry

{% highlight bash %} ServerName majorursa.net ServerAlias www.majorursa.net DocumentRoot /var/www/majorursalia/

A VirtualHost for a Drupal Site

{% highlight bash %} ServerName majorursa.net ServerAlias www.majorursa.net DocumentRoot /var/www/majorursalia/drupal-6.22 AllowOverride All Options None Order allow,deny Allow from all {% endhighlight %}
